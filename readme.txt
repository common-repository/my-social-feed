= Overview =

My Social Feed is the **Best Instagram Feed and Gallery management plugin for WordPress** built to display instagram feed in multiple ways on any post / page or widget! The plugin comes with the easiest Shortcode Generator, settings panel that can help you build awesome and unique instagram feed with responsive layouts and customizable styles on your website.

Since Instagram are so important to your potential customers, it is important that you showcase them effectively on your website and that you are regularly collecting fresh instagram feed. Instagram Pro helps you do both.


= Basic Input Fields =

* Instagram username or hasgtag 
* Gallery switcher
* Biography switcher 
* Profile switcher 
* Display IGTV

== Installation ==

= MINIMUM REQUIREMENTS =

* PHP 5.6 or later
* MySQL 5.6 or later
* WordPress 4.0 or later

**This section describes how to install the plugin and get it working**

= MANUAL INSTALLATION =

**Uploading in WordPress Dashboard**

* Download my-social-feed.zip
* Navigate to the ‘Add New’ in the plugins dashboard
* Navigate to the ‘Upload’ area
* Select my-social-feed.zip from your computer
* Click ‘Install Now’
* Activate the plugin in the Plugin dashboard

**Using FTP**

* Download my-social-feed.zip
* Extract the instagram-free directory to your computer
* Upload the instagram-free directory to the /wp-content/plugins/ directory
* Activate the plugin in the Plugin dashboard

You will now see the Instagram menu in your WordPress admin panel.

The WordPress codex contains <a href="https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation">instructions</a> on how to install a WordPress plugin.
